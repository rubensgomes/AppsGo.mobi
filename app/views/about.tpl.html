This is a non-commercial HTML 5 based web application site designed, optmized and implemented for viewing on mobile "smartphone" and tablets devices. It was developed making use of back-end HTTP REST calls to web services implemented in the Python programming language. The web services are located and accessible at RESTportal.com. The front-end pages were implemented using the JavaScript programming language using the AngularJS framework and JQuery Mobile librariers.
