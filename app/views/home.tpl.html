This site has been optimized for viewing on a mobile phone. It was created by Rubens Gomes.

This mobile site provides unit conversion utilities, currency rates, and a link to see an Internet of Things (IoT) project showing live room temperature sensor readings from Rubens' home.

This mobile site was developed using back-end REST APIs implemented in the Python programming language. The front-end pages were implemented using the JavaScript programming language, AngularJS framework, JQuery Mobile library, and HTML 5.
