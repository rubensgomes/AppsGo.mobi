Welcome

This is an experiemental mobile web application created by Rubens Gomes.

This application provides several unit conversion utilities, currency rates, and a link to see an IoT (Internet of Things) project showing live room temperature sensor readings from Rubens' home.

This mobile web app was developed using back-end REST APIs implemented in the Python programming language, and the front-end pages were implemented using AngularJS, JQuery Mobile, HTML 5 and the JavaScript programming language.

Open the sidebar menu and start testing features.
